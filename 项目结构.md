# 项目结构

## HIM 项目

这是一个包含后端和前端的项目。

### backend/ - 后端项目

这是一个基于 Spring Boot 的 Java 后端项目。

#### 目录结构

```
backend/
├── .mvn/
│   └── wrapper/
│       └── maven-wrapper.properties
├── logs/
│   ├── app.log
│   └── app.log.2026-06-10.0.gz
├── src/
│   └── main/
│       ├── java/
│       │   └── com/
│       │       └── example/
│       │           ├── annotation/          # 自定义注解
│       │           ├── common/              # 公共类
│       │           ├── config/              # 配置类
│       │           ├── controller/          # 控制器
│       │           ├── dto/                 # 数据传输对象
│       │           │   ├── request/         # 请求 DTO
│       │           │   └── response/        # 响应 DTO
│       │           ├── entity/              # 实体类
│       │           ├── enums/               # 枚举
│       │           ├── exception/           # 异常处理
│       │           ├── factory/             # 工厂模式
│       │           ├── filter/              # 过滤器
│       │           ├── repository/          # 数据仓库
│       │           ├── service/             # 服务层
│       │           │   └── impl/            # 服务实现
│       │           └── util/                # 工具类
│       └── resources/
│           └── application.yaml            # 配置文件
├── uploads/                                 # 上传文件目录
├── pom.xml                                  # Maven 配置文件
└── README.md
```

#### 主要功能模块

- **用户管理**: UserController, UserService
- **相册管理**: AlbumController, AlbumService
- **菜单管理**: MenuController, MenuService
- **权限管理**: PermissionController, PermissionService
- **手机管理**: PhoneController, PhoneService

#### 技术栈

- Spring Boot
- Spring Security + JWT
- Spring Data JPA
- OpenAPI (Swagger)

### frontend/ - 前端项目

这是一个基于 JavaFX 的桌面应用程序。

#### 目录结构

```
frontend/
├── .vscode/
│   └── settings.json
├── src/
│   └── main/
│       ├── java/
│       │   ├── com/
│       │   │   └── example/
│       │   │       ├── ai/                  # AI 服务
│       │   │       ├── cache/               # 缓存
│       │   │       ├── config/              # 配置
│       │   │       ├── controller/          # 控制器
│       │   │       ├── exception/           # 异常
│       │   │       ├── factory/             # 工厂
│       │   │       ├── menu/                # 菜单
│       │   │       ├── model/               # 模型
│       │   │       ├── service/             # 服务
│       │   │       └── util/                # 工具
│       │   └── io/
│       │       └── swagger/
│       │           └── client/              # Swagger 客户端代码
│       └── resources/
│           ├── com/
│           │   └── example/
│           │       ├── css/                 # 样式文件
│           │       └── view/                # FXML 视图文件
│           ├── icons/                       # 图标资源
│           └── db.properties                # 数据库配置
├── target/                                  # 编译输出目录
├── pom.xml                                  # Maven 配置文件
└── README.md
```

#### 主要功能模块

- **登录**: LoginController
- **首页**: HomeController
- **患者管理**: PatientManagementController
- **医生排班**: DoctorScheduleController
- **预约管理**: AppointmentManagementController
- **病历管理**: MedicalRecordController
- **药品管理**: MedicineManagementController
- **照片管理**: PhotoController
- **思维导图**: MindMapViewerController
- **快速生成**: QuickGenerateController

#### 技术栈

- JavaFX
- Swagger Codegen (API 客户端生成)
