# AI 思维导图 (AIMindMapWeb)

基于 JavaFX 的智能思维导图工具，采用前后端分离架构，AI 辅助生成和优化思维导图结构。

## ✨ 功能特性

- 🎨 **现代化界面**：基于 JavaFX 构建的跨平台桌面应用
- 🤖 **AI 智能生成**：输入主题，AI 自动生成思维导图结构
- 🔄 **前后端分离**：RESTful API 设计，便于扩展和维护
- 📊 **多种导图样式**：支持鱼骨图、树状图、组织结构图等
- 💾 **多格式导出**：支持 JSON、PNG、PDF、Markdown 等格式
- 🔍 **智能推荐**：基于主题的关键词和关联内容推荐

## 🛠️ 技术栈

### 前端 (JavaFX)
- **JavaFX 17+**：跨平台 UI 框架
- **OpenJFX**：JavaFX 开源实现
- **JFoenix**：Material Design 风格组件
- **GraphStream**：图形可视化库
- **OkHttp**：HTTP 客户端

### 后端
- **Spring Boot 2.7+**：后端框架
- **AI 服务**：集成 LLM API (OpenAI/Claude/国产大模型)
- **MySQL/PostgreSQL**：数据持久化
- **Redis**：缓存和会话管理
- **JWT**：身份认证

### 开发工具
- **Maven**：项目构建管理
- **Git**：版本控制
- **OpenAPI Generator**：API 客户端自动生成
