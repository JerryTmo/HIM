# JavaFX 桌面应用打包说明

本文档说明了如何将这个 JavaFX 项目打包成 Windows 桌面应用程序。

## 环境要求

- JDK 22 或更高版本（已配置 JAVA_HOME）
- Maven 3.6+（已配置到 PATH 环境变量）

## 打包步骤

### 1. 编译并打包项目

首先执行 Maven 编译和打包：

```bash
mvn clean package
```

### 2. 生成自定义 JRE 镜像

使用 javafx 插件的 jlink 目标来生成自定义 JRE：

```bash
mvn javafx:jlink
```

这会在 `target/javafx-demo` 目录下生成一个包含所需 Java 模块的自定义 JRE。

### 3. 使用 jpackage 创建安装程序

使用 jpackage 工具（包含在 JDK 16+ 中）创建 Windows 安装程序：

```bash
jpackage --name AIMindMap --input target --main-jar javafx-demo-1.0-SNAPSHOT-all.jar --main-class com.example.Launcher --runtime-image target/javafx-demo --dest target/dist --app-version 1.0.0 --vendor "AIMindMap Team" --win-menu --win-dir-chooser --win-per-user-install
```

或者，如果你安装了 `jpackage-maven-plugin`，可以直接使用：

```bash
mvn jpackage:jpackage
```

## 使用现有的打包成果

项目目录中已经有一个之前打包好的 `QQLogin` 目录，包含：
- `QQLogin.exe` - 可以直接运行的应用程序
- `app/` - 应用程序文件
- `runtime/` - Java 运行时环境

你可以直接运行 `QQLogin/QQLogin.exe` 来测试应用。

## pom.xml 已配置的功能

pom.xml 中已经配置了以下插件：

1. **javafx-maven-plugin**: 用于运行和 jlink 打包
2. **maven-jar-plugin**: 打包普通 JAR
3. **maven-dependency-plugin**: 复制依赖到 lib 目录
4. **maven-assembly-plugin**: 创建包含所有依赖的胖 JAR
5. **jpackage-maven-plugin**: （新增）用于创建桌面安装程序

## 其他平台打包

pom.xml 中已经配置了不同平台的 profile：

- Windows: `package-win`（默认激活）
- Linux: `package-linux`
- macOS: `package-mac`

要在特定平台上打包，可以使用对应的 profile。
